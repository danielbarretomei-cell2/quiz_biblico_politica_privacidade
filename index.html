Política de Privacidade — Quiz Bíblico

Última atualização: 20 de abril de 2026

O aplicativo Quiz Bíblico, desenvolvido por DR Technology, tem o compromisso de proteger a privacidade dos seus usuários. Esta política descreve como o app trata as informações.

1. Dados Coletados pelo App

O Quiz Bíblico não coleta, armazena nem transmite dados pessoais dos usuários para servidores externos. Todas as informações geradas pelo uso do app (nome do jogador, pontuação, conquistas, configurações) são armazenadas exclusivamente no dispositivo local do usuário, utilizando o mecanismo SharedPreferences do Android.

2. Funcionamento Offline

O app funciona 100% offline. Não é necessário criar conta, fazer login ou fornecer qualquer informação pessoal para utilizar o app.

3. Anúncios (Google AdMob)

O Quiz Bíblico exibe anúncios fornecidos pelo Google AdMob. O Google pode coletar e usar dados para personalizar anúncios de acordo com sua própria política de privacidade. Para mais informações sobre como o Google utiliza os dados, acesse:

https://policies.google.com/privacy

Os anúncios exibidos seguem as diretrizes do Google para apps voltados ao público geral.

4. ID de Publicidade

O app pode utilizar o ID de publicidade do Android (Advertising ID) fornecido pelo Google Play Services exclusivamente para fins de exibição de anúncios pelo AdMob. Esse ID pode ser redefinido ou desativado nas configurações do dispositivo Android.

5. Permissões do App

O app solicita as seguintes permissões:

  • INTERNET — necessária para carregar anúncios do AdMob
  • VIBRATE — para feedback háptico durante o quiz
  • POST_NOTIFICATIONS — para enviar lembretes diários opcionais
  • RECEIVE_BOOT_COMPLETED — para reagendar notificações após reinicialização
  • SCHEDULE_EXACT_ALARM — para notificações de lembrete no horário configurado

Nenhuma dessas permissões é usada para coletar dados pessoais.

6. Notificações Locais

O app pode enviar notificações locais de lembrete diário para jogar. Essas notificações são geradas localmente no dispositivo e podem ser desativadas a qualquer momento nas configurações do app ou nas configurações do Android.

7. Crianças

O Quiz Bíblico é um app educativo adequado para todas as idades. Não coletamos intencionalmente dados de crianças menores de 13 anos. Como o app não coleta nenhum dado pessoal, ele é seguro para uso por crianças sob supervisão dos pais.

8. Alterações nesta Política

Esta política pode ser atualizada periodicamente. Quaisquer alterações serão publicadas nesta página. Recomendamos verificar esta política regularmente.
